import { Tabs, TabItem } from '@astrojs/starlight/components';

:::danger[Important]
Be aware that the execution of the commands below depends on the projects. Some may be missing in some projects while all of these commands may work in others.

Please consider they do indeed exist in their respective configuration files:

> - `pnpm`: See **package.json** file
> - `Composer`: See **composer.json** file
> - `Make`: See **Makefile**
:::

## 🧹 Keep a modern codebase

<Tabs>
<TabItem label="pnpm">

```sh
pnpm check
```

</TabItem>
<TabItem label="Composer">

```sh
composer check
```

</TabItem>
<TabItem label="Make">

```sh
make check
```

</TabItem>
</Tabs>
